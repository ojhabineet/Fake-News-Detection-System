# Fake-News-Detection-System
A natural language processing (NLP) based system that classifies news articles as Fake or Real.

Tech Stack:

Python
Pandas
NLP (TF-IDF)
Scikit-learn
Logistic Regression

Workflow:

Text cleaning & preprocessing
TF-IDF feature extraction
Model training
Fake/Real news prediction

Use Case:
Social media monitoring and misinformation detection
